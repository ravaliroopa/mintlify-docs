**Best for:** Hands-off traders who want AI to find and trade opportunities across platforms.

This agent uses an AI model (GPT-4) to scan markets, research events using web search, and make trading decisions autonomously.

## How It Works

1. **Market Scanner**: Periodically scans markets on your chosen platform
2. **News Monitor**: Monitors news feeds for market-moving events
3. **AI Decision**: Uses an LLM to analyze each opportunity, reads news, assesses probability
4. **Trade Execution**: Places trades when confidence exceeds your threshold
5. **Stop-Loss Monitor**: Watches positions and triggers stop-losses when needed

## Supported Platforms

- Polymarket
- Kalshi
- Limitless

## Settings

| Setting | Description | Default |
|---------|-------------|---------|
| **Platform** | Which exchange to trade on | `polymarket` |
| **Position size** | USD per trade | $5 |
| **Max daily trades** | Limit trades per day | 5 |
| **Max open positions** | Concurrent position limit | 3 |
| **Stop-loss** | Auto-close losing positions | 10% |
| **Confidence threshold** | Minimum AI confidence to trade | 80% |
| **Categories** | Limit to specific categories | All |
| **Excluded markets** | Blacklist specific markets | None |
| **Research depth** | AI research thoroughness | Advanced |

<Warning>
  The autonomous agent uses AI decision-making, which is not infallible. Always set appropriate position sizes and stop-losses.
</Warning>
