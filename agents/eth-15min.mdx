**Best for:** Traders who want AI-powered ETH price predictions.

Trades ETH 15-minute markets on Polymarket using Allora Network AI predictions.

## How It Works

1. Every 15 minutes, a new market opens: "Will ETH go up or down?"
2. The agent fetches an AI price prediction from the **Allora Network**
3. Compares the prediction to the current ETH price (via Chainlink oracle)
4. If AI predicts UP, buys **Yes**. If DOWN, buys **No**.

## Strategies

| Strategy | Description |
|----------|-------------|
| **Up/Down** (default) | Trades both up and down predictions |
| **Up Only** | Only trades when the AI predicts ETH will go up (more conservative) |

## Allora Network

The agent uses the [Allora Network](https://allora.network), a decentralized AI inference network that aggregates predictions from multiple AI models.
